HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Inline CSS is when a developer wants to style an element directly in the HTML file.
  • Internal style sheet contains CSS rules for the webpage in the head section of the HTML file.
  • External CSS style sheet allows developers to keep all their CSS rules in a separate file.
  • A selector defines the element or attribute to which the rules, or declarations, will apply.
  • Use this specail wild card symbol * to apply a rule to all elements visible on the page.
  • DRY code stands for Do Not Repeat Yourself.
  • Test-align is a positioning property that can help center a title or heading.
  • An img selector assigns property vaules to any image element in the markup.
  • A link element creates a connection to an external file.
  • A rel attribute specifies the relationship between the current documentation and the linked document resource.
  • A href attribute specifies the location URL of the external resource.
  • box-shadow CSS property adds a shadow effect around an elements frame.
